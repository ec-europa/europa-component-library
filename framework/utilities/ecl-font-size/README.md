# Font size utilities

Use the following classes to quickly set a font size:
* **ecl-fs-xxl**: Extra extra large text
* **ecl-fs-xl**: Extra large text
* **ecl-fs-l**: Large text
* **ecl-fs-m**: Medium text
* **ecl-fs-s**: Small text
* **ecl-fs-xs**: Extra small text
* **ecl-fs-xxs**: Extra extra small text
