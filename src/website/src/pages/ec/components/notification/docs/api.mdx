## Setup

There are 2 ways to initialise the component.

### Automatic

Add `data-ecl-auto-init="Notification"` attribute to component's markup:

```html
<div
  role="alert"
  class="ecl-notification ecl-notification--info"
  data-ecl-notification
  data-ecl-auto-init="Notification"
>
  ...
</div>
```

Use the `ECL` library's `autoInit()` (`ECL.autoInit()`) when your page is ready or other custom event you want to hook onto.

### Manual

Get target element, create an instance and invoke `init()`.

Given you have 1 element with an attribute `data-ecl-notification` on the page:

```js
var elt = document.querySelector('[data-ecl-notification]');
var notification = new ECL.Notification(elt);
notification.init();
```
