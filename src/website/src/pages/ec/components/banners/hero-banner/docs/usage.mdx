  A Hero banner displays the main message of the page, along with additional
  information and a clear action, if needed. They are a good way to inform users
  of the main goal of a certain page, such as registering to an event or
  applying for funding.

### Do's

- always use with a short, distinct and indicative heading
- keep in mind the font-to-background contrast ratio, this improves a page's readability (e.g. dark backgrounds - white text / light backgrounds - black text)

### Don'ts

- do not use the content-width variant next to a full-width element (such as the header & footer)
- do not display more than 1 call to action per page
- do not use more than 3 banners on the same page (top, middle and bottom)
- do not opt for center font-alignment, unless you already have center-aligned content

### When to use it

- when you need to promote or convey an important message to user
- exclusively on Harmonised (group 2) sites

### When not to use it

- do not use it on Core, Standardised and Harmonised (group 1 & 3) sites - use <Link to="/ec/components/banners/page-banner/usage/">Page banner instead</Link>

#### Default

| Elements            | Mandatory | Description                                                                                                                                                                                                                                                                             |
| ------------------- | --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| image               | yes       | representative image for the page content<br /><br />**Note**: aspect ratio of 4:1                                                                                                                                                                                                      |
| heading             | yes       | heading of the item<br /><br />**Note**: limited to 50 characters, including space - limitation seen on the CMS, adding it now might impact the usage page of other components                                                                                                          |
| description         | no        | a summary of the content that the component addresses<br /><br />**Note**: limited to 150 characters, including space - limitation seen on the CMS, adding it now might impact the usage page of other components                                                                       |
| call to action link | no        | link to either internal or external source (the call to action label can be changed to indicate the intended action or goal)<br /><br />**Note**: limited to 25 characters, including space - limitation seen on the CMS, adding it now might impact the usage page of other components |

#### Conditional

| Elements                | Mandatory | Description                                                                                                                                                                                                                                 |
| ----------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| text box/shaded overlay | yes       | element that allows for better font-to-background contrast ratio<br /><br />**Note**: If the shaded option has been selected, the text will then be displayed directly on top of the image and not within the text box (see anatomy images) |

### Do's

- always use with a short, distinct and indicative heading
- select an appropriate image, that is of high quality, suggestive, complementary and relevant for the related content
- choose the appropriate version (Text-box or Shaded), depending on the chosen image while keeping the contrast ratio in mind
- keep in mind the font-to-background contrast ratio, this improves a page's readability (e.g. dark backgrounds - white text / light backgrounds - black text)
- make sure the highlight or main object within the image is visible (depending on the alignment option selected)

### Don'ts

- do not use the content-width variant next to a full-width element (such as the header & footer)
- do not chose an image that could be visually intrusive or overwhelming in terms of elements
- do not display more than 1 call to action per page
- do not use more than 3 banners on the same page (top, middle and bottom)
- do not opt for centre font-alignment, unless you already have centre-aligned content

### When to use it

- when you need to promote or convey an important message to user
- exclusively on Harmonised (group 2) sites

### When not to use it

- do not use it on Core, Standardised and Harmonised (group 1 & 3) sites - use <Link to="/ec/components/banners/page-banner/usage/">Page banner instead</Link>
