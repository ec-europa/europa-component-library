## Setup

There are 2 ways to initialise the component.

### Automatic

Add `data-ecl-auto-init` attribute to component's markup

```html
<nav
  class="ecl-menu-legacy"
  data-ecl-menu-legacy
  data-ecl-auto-init="MenuLegacy"
>
  ...
</nav>
```

Use the `ECL` library's `autoInit()` (`ECL.autoInit()`) when your page is ready or other custom event you want to hook onto.

### Manual

Get target element, create an instance and invoke `init()`.

Given you have 1 element with an attribute `data-ecl-menu-legacy` on the page:

```js
var elt = document.querySelector('[data-ecl-menu-legacy]');
var menuLegacy = new ECL.menuLegacy(elt);
menuLegacy.init();
```
