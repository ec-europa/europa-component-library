<Paragraph size="lead">
  Tags are used to provide contextual information; find related content using
  keywords, labels or links. Tags are also used as keywords in search boxes,
  filters and category lists. In some cases they can be added or removed by the
  user.
</Paragraph>

## Anatomy

There's 2 variant of tags to accommodate different needs

### Link tags

| Name     | Constraint |
| -------- | ---------- |
| Label    | Mandatory  |
| URL-link | Mandatory  |

### Removable tags

| Name         | Constraint |
| ------------ | ---------- |
| Label        | Mandatory  |
| Dismiss icon | Mandatory  |

## When to use it

### Link tags

- To navigate to another page/content
- Use tags to link current content to other related content
- showing information which helps a user make sense of the content on the page, such as filter parameters or object metadata.
- Use to display information that clarifies the content on the page, such as metadata.
- Use to enable the tag text to navigate to other pages or additional information.
- Use tags to showcase the specific attributes of an item. 
- Use tags to display applied filters or options

### Removable tags

- To dismiss a tag from a search query (ex:search filter, facets...)

## When not to use it

### Link tags

- Avoid using too many tags as they might cause a visual overload.
- Avoid using tags that are too long (more than 3 words).
