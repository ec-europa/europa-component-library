## Do's

- **always use a short, distinct and indicative heading**

## Don'ts

- **do not duplicate this composition within a single page**

## When to use

- use whenever you have News, Events and Publications and want to display them on a page

## When not to use

- this is a bespoke solution - **do not use for anything else, except News, events and publications on a page**
