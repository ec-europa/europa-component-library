## Anatomy

A media container composition contains:

- heading 2
- image or a video and a caption
- paragraph title
- paragraph
- link / Call to action link

| Elements                   | Mandatory | Description                                 |
| -------------------------- | --------- | ------------------------------------------- |
| heading 2                  | no        |                                             |
| image or video             | yes       | left or right aligned                       |
| caption                    | no        | explanatory caption to complement image     |
| paragraph title            | no        |                                             |
| paragraph                  | yes       | simple rich text                            |
| link / call to action link | no        | default label is "read more" (overwritable) |

## When to use

- people respond positively to visual content. Having a site-wide horizontal image on top of your website is a good way to invite visitors to explore your website, or a particular section of it.
