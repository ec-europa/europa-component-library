<Paragraph size="lead">
  Screen reader utilities let you let you add content visible only to screen
  reading devices.
</Paragraph>

Available classes are:

- `.ecl-u-sr-only`: always hidden on screen, and available on screen reader
- `.ecl-u-sr-only--focusable`: hidden by default on screen, but can be displayed if focused (with keyboard mostly)
