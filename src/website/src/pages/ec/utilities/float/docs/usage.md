<Paragraph size="lead">
  Float utilities let you put element as floating (left or right), or cancel
  current floating status.
</Paragraph>

Available classes are:

- `.ecl-u-f-l`: float left
- `.ecl-u-f-r`: float right
- `.ecl-u-f-none`: float none
