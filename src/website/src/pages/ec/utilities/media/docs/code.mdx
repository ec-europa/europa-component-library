Use following classes to alter media display.

Examples display media using `img` tag on the left and
`background-image` on the right.

**Important:** Depending on the clases used, image may be distorded or not correctly displayed (see examples below). So these utilities have to be used wisely.

## Media sizes

<Playground system="ec" selectedKind="utilities-media" selectedStory="default">
  <Media type="a" token="s" />
  <Media type="a" token="m" />
</Playground>

## Horizontal size only

<Playground system="ec" selectedKind="utilities-media" selectedStory="default">

  <Media type="h" token="m" />
</Playground>

## Vertical size only

<Playground system="ec" selectedKind="utilities-media" selectedStory="default">
  <Media type="v" token="s" />
  <Media type="v" token="m" />
</Playground>

## Responsive size

Add the breakpoint before media size to display it only with corresponding screen size (works for all kind of size ahead).

Examples:

- media `m` on screen size >= breakpoint `md`, and `s` otherwise:  
  `ecl-u-media-a-s ecl-u-media-a-md-m`
- horizontal size `m` on screen size >= breakpoint `lg`, and `s` otherwise:  
  `ecl-u-media-h-s ecl-u-media-h-lg-m`

## Ratio

To set ratio of an image, wrap it with a ratio class and add class `ecl-u-media-content` to the image.  
Note it does make sense only if one of the image dimension only are fixed (width or height).  
Behavior is the same for `img` and `background-image`.

**Important:** when using ratio, media size and margin/padding have to be put on the ratio wrapper, not on the media itself.

<Playground system="ec" selectedKind="utilities-media" selectedStory="default">
  <h4 className="ecl-u-type-heading-4">Fixed width</h4>
  <MediaRatio type="h" token="m" ratio="1-1" />
  <MediaRatio type="h" token="m" ratio="16-9" />
  <h4 className="ecl-u-type-heading-4">Fixed height</h4>
  <MediaRatio type="v" token="m" ratio="1-1" />
  <MediaRatio type="v" token="m" ratio="16-9" />
</Playground>
