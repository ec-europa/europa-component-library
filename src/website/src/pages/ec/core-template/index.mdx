## Structural anatomy

When designing the page, make sure to follow appropriate category guidelines available below in terms of structure and layout

| Elements                       | Mandatory  | Description                                               |
| ------------------------------ | ---------- | --------------------------------------------------------- |
| <Link to="#">Grid</Link>       | Yes        | Enhance visual consistency                                |
| <Link to="#">Colours</Link>    | Yes        | Use only EC colour specifications available in the ECL    |
| <Link to="#">Typography</Link> | Yes        | Apply the typographic guidelines available in the ECL     |
| <Link to="#">Spacing</Link>    | Yes        | Apply spacing rules explained in the ECL                  |
| <Link to="#">Icons</Link>      | Contextual | Where icons feature, use commonly-used icons from the ECL |
| <Link to="#">Images</Link>     | Contextual | See image guidelines in ECL                               |

## Do's

- ensure the pages contain all the elements in the Anatomy table above

## Don'ts

- do not replace pages only on existing websites as this will create visual inconsistencies, this must be done at a site level

## When to use

- when updating pages on the Core site (first three levels of ec.europa.eu)

## When not to use

- do not follow these guidelines when you are updating sites that fall under the <Link to="#">standardised</Link> or <Link to="#">harmonised</Link> category
