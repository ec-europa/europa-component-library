<Paragraph size="lead">
  The department content type is an adaptable concept for any type of entity.
  This can be used for types of organisational entities such as sectors, teams,
  units, working groups, etc. It provides our users with an overview of the
  responsibilities of a certain organisational entity, information on the
  leadership structure and organisation, plans, reports and supporting documents
  as well as the contact details and links to related pages.
</Paragraph>
