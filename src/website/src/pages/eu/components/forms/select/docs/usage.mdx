  displays a list of selectable items from which the user can select one value.

### Default

| Elements         | Mandatory | Description                                                                                                                     |
| ---------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------- |
| Heading          | Yes       | Heading describes the options in the Select Menu                                                                                |
| Helper text      | No        | Helper text information assists a user in using Select Menu                                                                     |
| Dropdown List    | Yes       | Dropdown in the collapsed state with placeholder giving instructions for using the Dropdown Select                              |
| Action indicator | Yes       | Varies depending on the state (collapsed/expanded). Indicates what the next action will do - **displays listed items on click** |

### Conditional (missing input)

| Elements           | Mandatory | Description                                                             |
| ------------------ | --------- | ----------------------------------------------------------------------- |
| Optional indicator | Yes       | Indicator stating wether **if** input fields are **optional**           |
| Error message      | Yes       | A message informing the user **if** there is a problem with their entry |

## Do's

- write short, distinct and indicative headings for the **items listed inside the dropdown**
- indicate whether the text field is **optional**
- **make use of helper text** if there are further directions or hints the users may need in completing their goal (example: _A password must be at least 6 characters and contain a number and a special character_)
- **write specific and clear error message texts, so users understand how to address the error**

## Don'ts

- **do not use as navigation**
- do not use when there are very few items for which radio buttons could be used

## When to use

- when there are more than 10 options to select from

## When not to use

- when the options in the select menu extend beyond the page height
