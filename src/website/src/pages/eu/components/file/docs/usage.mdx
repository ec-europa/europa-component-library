### Default

| Elements    | Mandatory | Description                                                 |
| ----------- | --------- | ----------------------------------------------------------- |
| Title       | Yes       | Title must describe the downloadable file                   |
| Details     | Yes       | Details displays the file language, file type and file size |
| Link Button | Yes       | Text Link downloads the file when selected                  |

### Conditional (translated/multiple languages)

| Elements       | Mandatory | Description                                                                                                      |
| -------------- | --------- | ---------------------------------------------------------------------------------------------------------------- |
| Expand Button  | Yes       | Text button, when selected, displays the file in different languages available, **below the download container** |
| Language       | Yes       | The language of the downloaded content **acting as a title**                                                     |
| Helper Message | No        | Helper message assists the user in selecting the appropriate file and language                                   |

## Do's

- <strong>label each item</strong> with a short, distinct and indicative title
- always include the <strong>file language</strong>, <strong>file type</strong> and <strong>file size</strong>

## Don'ts

- don't use without additional relevant content on the page
- don't use the same title as the file name

## When to use

- when you want to include additional relevant or supporting documents for download

## When not to use

- Do not use to include <strong>important information</strong> that should be present at all time
