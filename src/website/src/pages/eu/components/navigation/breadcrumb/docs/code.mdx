## Long breadcrumb

When there are more than 3 items, the breadcrumb should be collapsed by default and items should be displayed if there's enough space.

The first item and the last 2 items should always be visible.

<Playground
  system="eu"
  selectedKind="navigation-breadcrumb"
  selectedStory="long"
>
  <div style={{ backgroundColor: '#004494' }}>
    <Breadcrumb label={demoContent.label} ellipsisLabel="Click to expand">
      {demoContent.items.map(item => (
        <BreadcrumbItem {...item} key={item.label} />
      ))}
    </Breadcrumb>
  </div>
</Playground>
