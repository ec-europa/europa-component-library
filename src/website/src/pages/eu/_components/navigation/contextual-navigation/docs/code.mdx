## JS behaviour

### Auto initialisation

In order to automatically initalise the JS behaviour, add `data-ecl-auto-init="ContextualNavigation"` to your component's markup:

```html
<nav
  class="ecl-contextual-navigation"
  data-ecl-contextual-navigation
  data-ecl-auto-init="ContextualNavigation"
>
  ...
</nav>
```

Don't forget to call `ECL.autoInit()` when your page is ready!

### Manual initialisation

When the document is ready, query the element. For example, you can use `document.querySelector('[data-ecl-contextual-navigation]')` if you only have 1 contextual navigation in the page.

Then, instantiate the `ContextualNavigation` component and call `init()`:

```js
var elt = document.querySelector('[data-ecl-contextual-navigation]');
var contextualNavigation = new ECL.ContextualNavigation(elt);
contextualNavigation.init();
```
