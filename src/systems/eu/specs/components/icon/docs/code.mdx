You can pick an icon from <Link to="/eu/guidelines/iconography/">the library we provide</Link>.

The ID of the icon is prefixed with its set: for example, the ID of the "audio" icon from the "general" set is `general--audio`.

Note: display of icons on Internet Explorer is made possible by the use of [svg4everybody](https://github.com/jonathantneal/svg4everybody)

## Color

You can use the default, primary or inverted colours:

<Playground
  system="eu"
  selectedKind="Icon"
  selectedStory="default"
>
  <Icon {...demoContentAudio} />
  <Icon {...demoContentAudio} color="primary" />
  <Icon {...demoContentAudio} color="inverted" />
</Playground>

## Size

You can change the size of the icon by applying one of the following classes:

- `ecl-icon--xs`: 16&times;16
- `ecl-icon--s`: 20&times;20
- `ecl-icon--m`: 24&times;24
- `ecl-icon--fluid`: 1em

<Playground
  system="eu"
  selectedKind="Icon"
  selectedStory="default"
>
  <Icon {...demoContentAudio} size="xs" />
  <Icon {...demoContentAudio} size="s" />
  <Icon {...demoContentAudio} size="m" />
  <Icon {...demoContentAudio} size="fluid" />
</Playground>

## Rotate icons

You can rotate icons by adding one of the following classes:

- `ecl-icon--rotate-0`: 0 degree
- `ecl-icon--rotate-90`: 90 degrees
- `ecl-icon--rotate-180`: 180 degrees
- `ecl-icon--rotate-270`: 270 degrees

<Playground
  system="eu"
  selectedKind="Icon"
  selectedStory="default"
>
  <Icon {...demoContentAudio} transform="rotate-0" />
  <Icon {...demoContentAudio} transform="rotate-90" />
  <Icon {...demoContentAudio} transform="rotate-180" />
  <Icon {...demoContentAudio} transform="rotate-270" />
</Playground>

## Flip icons

You can flip icons by adding the class `ecl-icon--flip-horizontal` or `ecl-icon--flip-vertical`).

<Playground
  system="eu"
  selectedKind="Icon"
  selectedStory="default"
>
  <Icon {...demoContentAudio} transform="flip-horizontal" />
  <Icon {...demoContentAudio} transform="flip-vertical" />
</Playground>
